# 🎮 CS2 Map Picker

Простой и удобный инструмент для выбора (пика/бана) карт из маппула Counter-Strike 2.  
Кликните по карте — она зачеркнётся. Поддерживает добавление, удаление и сброс карт. 

## ✨ Возможности

- 🃏 **10 карт CS2** (Dust2, Mirage, Inferno, Vertigo, Train, Anubis, Ancient, Ancient Night, Overpass, Nuke)
- 🖼️ **Квадратные превью** карт с названиями под ними
- ✅ **Зачёркивание по клику** — красный крест и серый текст (повторный клик снимает)
- 📋 **Редактируемый список карт** через файл `maps.txt` (можно менять вручную или через интерфейс)
- ➕ **Добавление новых карт** — введите название, программа обновит список
- ❌ **Удаление карт** — правый клик для выделения, затем кнопка «Удалить»
- 🔄 **Сброс к стандартному пулу** одной кнопкой
- 🧹 **Сброс всех зачёркиваний**
- 🚀 **Встроенные изображения** — картинки уже внутри EXE, не нужно таскать папку с файлами (но можно заменить своими, положив рядом `.jpg`)
- 💻 **Кроссплатформенность** — работает на Windows, Linux, macOS (требуется Python и Pillow)


## 📦 Скачать готовый EXE (Windows)

Перейдите в [Releases]([https://github.com/yourusername/cs2-map-picker/](https://github.com/pkrff/map-pick-cs2-map_picker-)/releases) и скачайте последнюю версию `CS2_MapPicker.exe`.  
Запустите — всё готово к работе. При первом запуске автоматически создастся файл `maps.txt` со стандартным списком.
